Biz Front-End kursida 6-oy Ichida Nimalarni organdik?

1 - HTML

bu veb-sahifalarni yaratish va tuzish uchun ishlatiladigan markirovka tilidir. HTML yordamida siz sahifadagi matn, rasmlar, havolalar, jadval va boshqa elementlarni tashkil etishingiz mumkin

2 - CSS

bu HTML hujjatlarining ko‘rinishini va dizaynini boshqarish uchun ishlatiladigan uslubni belgilash tili. CSS yordamida siz HTML elementlariga rang, shrift, o‘lcham, joylashuv va boshqa vizual xususiyatlarni qo‘shishingiz mumkin

3 - Tailwind CSS

bu utility-first yondashuviga ega, zamonaviy va minimalist CSS ramkasi. Tailwind CSS sizga HTML elementlarini tezda va intuitiv usulda uslublash imkonini beradi, bunda ko'plab kichik utility sinflarini ishlatish orqali murakkab dizaynlar yaratish mumkin

4 - JavaScript

bu veb-sahifalarga dinamiklik, interaktivlik va funksionallik qo‘shish uchun ishlatiladigan dasturlash tilidir. JavaScript veb-brauzerda ishlaydi va server tomonida ham ishlatilishi mumkin
